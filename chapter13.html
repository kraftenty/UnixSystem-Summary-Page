Chapter 13

Basic Shell Scripting

Script Language

  • Script is a program runs on the interpreter.
  • Unix Shell is an interpreter.
    DOS shell is an interpreter, too.
  • Shell reads the script and runs it directly line-by-line.
    compare to compile language like C/C++, must be compiled before running.

Usage

  • Shell program(scripts) is used for system administration
  • Automate a vast of range of jobs with a few simple scripts (Boring and repetitive jobs)

Script file

  • #!/bin/sh
    - Executed by /bin/sh
    - Shebang : hash(#), bang(!)
  • # : comments
  • variable=value
    - a=1
    - a=`expr $a+1` //back quote
  • $ read a : read from keyboard and save to variable a
    3
    $ echo $a
    3

Special Variables

  • ? : previous command's exit status.
    $ echo $?
    0 : exit normally
    1 : exit with problem
  • $ : PID of the current shell
    $ echo $$
    1234 (PID)
  • - : Options invoked at start-up of the current shell.
  • ! : PID of the last background process.
  • 0 : script name
  • 1-9 : command-line arguments 1-9.
  • _ : The last argument given to the most recently invoked command before this one.

Flow Control

  • Test Command
    - test expression
    - [ expression ]
  • Operator
    [ a -eq b ] : a==b
    [ a -ne b ] : a!=b
    [ a -gt b ] : a>b
    [ a -ge b ] : a>=b
    [ a -lt b ] : a<b
    [ a -le b ] : a<=b
    [ ! a ] : !a
    [ a -a b ] : a && b
    [ a -o b ] : a || b
  • Conditional Flow Control
    if-then-fi
    if-then-else-fi
    case-in-esac
  • Iterative Flow Control
    while-do-done
    until-do-done
