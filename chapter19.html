Installing Software from Source Code

Open Source Licensing

  • BSD License
    - Source code can be modified
    - Modification do not have to be redistributed
    - Advertised as BSD code is used
    - Used as proprietary or commercial software
  • GNU Public License (GPL)
    - Called as "copylight license"
    - Any modified version of the distributed software can be released unter the same GPL terms
    (-> termed as not "open source" but "free software")

Compile C Program


  • make : the program that runs commands
  • makefile : the file that specifies commands
  • -c : compile and make object file
  • -o : set output file's name

  • $ gcc hello.c
  • $ gcc -o hello hello.c
  • $ gcc -c main.c sub1.c sub2.c
  • $ gcc -o maketest main.o sub1.o sub2.o
  • make
    - makefile, Makefile
    - make -f make.my
    - make <target>
