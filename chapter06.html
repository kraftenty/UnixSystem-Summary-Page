Chapter 06

Unix Commands In-Depth

Anatomy of a Command

  1. $ command [options] [arguments] []:optional. not a mendatory.
  2. - $ ls -sCF /usr/bin
  3. - $ ls -s -C -F /etc
  4. $ man ls : print manual about ls command
  5. $ apropos ls : find manual with explanation related to ls.
  6. $ whereis echo : find direction of manual's execute file, source file, manual file.
  7. $ which echo : find echo command's direction

Meta Characters

  1. WildCard
  2. ? : matches any one character
  3. * : matches character or characters
  4. [] : matches one of the characters in [...]

  5. $ ls *.txt : print all text files
  6. $ ls hi?.txt : print hia.txt, hi2.txt, hi3.txt,...
  7. $ ls hi[12357].txt : print hi1.txt, hi2.txt, hi3.txt, hi5.txt, hi7.txt

Redirection

  1. $ ls > ls.txt : save result of ls command in ls.txt
  2. $ sort < /etc/group : sort and print the /etc/group file.
  3. $ sort < /etc/group > sort.txt : save the sorted /etc/group file in sort.txt
  4. $ mkdir dir1, $ mkdir dir1 2> err.txt : save the error(file exists) in err.txt
  5. $ find / -name 'hi.txt' 2> /dev/null : find hi.txt on root directory, and delete error messages into null device.
  6. $ find /export/home -name 'hi.txt' > out.txt 2>&1 : find hi.txt on /export/home directory, and save output to out.txt and save error message to out.txt also.
  7. $ find /export/home -name 'hi.txt' 2> out.txt 1>&2 : find hi.txt on /export/home directory, and save error message to out.txt and save output to out.txt also.
  8. $ mailx khlee < hi.txt : send mail to khlee with hi.txt's text.

Pipes

  1. $ who | wc : count word who command
  2. $ cat /etc/passwd | grep unix30 : print information with unix30 at passwd file.
  3. = $ grep unix30 /etc/passwd
  4. $ echo $PATH : print PATH's value
  5. $ echo ${PATH}${TERM} : print PATH's value and TERM's value
  6. $ echo "$PATH" : print PATH's value
  7. $ echo '$PATH' : print $PATH literally.
  8. $ echo "\$PATH" : print $PATH literally.
  9. $ echo today date is `date` : `date` is replaced by date command's result.
