Chapter 01

Unix Basic

We can remote connect in unix system by using telnet and ssh. And also we can use Putty program for convenience.

For connecting with Unix server, we should go to windows setup and check telnet client program to enable telnet at dos shell.
Or we can download Putty program and connect with Unix server via Putty program.
In Putty program, we can use telnet and SSH.

Unix command syntax consists of 3 parts. 1. command, 2. option, 3. arguments.
For example, $ ls -sCF /usr/bin.
ls command prints contents in current or particular directory.
-sCF is options. -sCF is same with -s -C -F. -s option show contents with size. -C option show contents by columns. -F option shows file type.
/usr/bin is argument. It represents path.
Option and arguments in command is optional. So we can execute command without options and arguments.
For example, $ ls shows current directory's contents. and $ ls / shows root directory's contents. and $ ls -F shows current directory's contents with file type.
