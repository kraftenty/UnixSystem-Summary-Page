Chapter 07

VI

VI is text editor that included in unix system.
We can execute vi by $ vi. And we can open or create file by $ vi filename.
Vi has two modes : command mode and insert mode.
Users can input command and moving,editing,saving,reading by using Command mode.
And users can insert characters by using Insert mode.
- command mode to insert mode : a, i, o
- insert mode to command mode : esc

Moving within a file

  1. h,j,k,l : left,down,up,right. you can do same thing with arrow keys.
  2. 0 or ^ : go to left most of the line.(same with home key)
  3. $ : go to right most of the line.(same with end key)
  4. [ENTER] : next line
  5. [ctrl+f] : one page forward (same with pgup key)
  6. [ctrl+b] : one page backward (same with pgdn key)
  7. [ctrl+u] : half page up
  8. [ctrl+d] : half page down
  9. :0 or :1 : go to first line
  10. :$ : go to last line
  11. :100 : go to the 100th line
  12. [ctrl+5] : moving cursor between bracket

Searching within a file

  1. /string : search string(forwards). ex) /hi= search hi within file.
  2. n :go to next string, N: go to previous string. (during searching)
  3. ?string : search string(backwards).

Exit and Save

  1. :q : quit(asks whether saving or not)
  2. :w : write to the file (save)
  3. :wq : write and quit
  4. :q! : exit without asking and saving
  5. :w! : overwrite
  6. :wq! : overwrite and exit
  7. ZZ : save and exit promptly
  8. x : exit
  9. w filename : write to file filename
  10. r filename : read filename file and paste at the current file.

Insert mode

  1. i : insert
  2. a : append
  3. o : insert from next line
  4. [esc] : go to command mode.

Delete

  1. x : delete a character
  2. 5x : delete 5 character
  3. dd : delete 1 line
  4. 3dd : delete 3 lines
  5. D : delete string left of the cursor in line.
  6. J : join current line and next line.
  7. i -> [ENTER] -> [ESC] : seperate line

Change

  1. r : replace one character. ex) ra : 1 character is changed to 'a'.
  2. 3rc : replace 3 characters in 'c'.
  3. cwWORD [ESC] : change 1 word to WORD.
  4. 3cwWORD [ESC] : change 3 words to WORD.

Undo / Redo

  1. u : undo
  2. uu : cancel undo
  3. . : redo

Copy and move

  1. How to copy a line==> yy -> [move cursor] -> p
  2. How to copy several lines==> 3yy -> [move cursor] ->p
  3. How to move a line==> dd -> [move cursor] -> p
  4. How to move several lines==> 3dd -> [move cursor] -> p

Running Commands

  1. :%s/abc/ABCD/g : change all 'abc' to 'ABCD' in whole file.
  2. :0,$ s/unix/UNIX/g : change all 'unix' to 'UNIX' in file start to end.
  3. :10,$ s/abc/ABCD/g : change all 'abc' to 'ABCD' from line 10 to the end of file.
  4. :100,200 s/unix/UNIX/g : change all 'unix' to 'UNIX' from line 100 to line 200.
  5. :100,200 w! filename : save line 100 to 200 to new file filename.

Miscellaneous commands

  1. :set nu : show line number
  2. :set nonu : hide line number
  3. :sh : go to shell temporary. return to vi by $ exit.
  4. :!pwd : run a command in vi. press [ENTER] to edit continuously.
  5. [Ctrl+l] : reset screen.
