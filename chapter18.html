Chapter 18

Backup Tools

Backup Media Types

  • Magnetic Tapes
  • CD/DVD-ROM
  • Zip drives
  • Hard drive / network backup
  • USB

Backup Types

  • Full Backup
    - entire
  • Differential Backup
    - that has changed since last full backup
  • Incremental Backup
    - that has changed since the completion of any other type of backup

tar

Tape archive
  • c : create(bind)
  • t : shows contents of bound file's path and information
  • x : extracts
  • v : verbose(print procecure when binds or extracts)
  • f : set filename
  • $ tar cvf mydir.tar mydir : bind mydir to mydir.tar
  • $ tar tvf mydir.tar : show contents of mydir.tar's path and information
  • $ tar xvf mydir.tar : extract mydir.tar

compresss

  • $ compress mydir.tar
    -> mydir.tar.Z
  • $ uncompress mydir.tar
    (= $ uncompress mydir.tar.Z)
  • $ gzip mydir.tar
    -> mydir.tar.gz
  • $ gzip -d mydir.tar.gz
    (= $ gunzip mydir.tar.gz)
    (= $ gunzip mydir.tar)

example

$ tar cvf mydir.tar mydir
$ compress mydir.tar
$ ftp 10.10.54.75
Login using id/passwd
ftp> bi
ftp> put mydir.tar.Z
ftp> quit
$ telnet 10.10.54.75
Login using id/passwd
$ uncompress mydir.tar.Z
$ tar xvf mydir.tar
