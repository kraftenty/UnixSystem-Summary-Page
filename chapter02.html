Chapter 02

Unix Basic Commands

This is Unix system basic commands.

$ ls : shows contents in directory.
Simply ls shows contents in current directory, and ls /usr/bin shows contents in /usr/bin directory.

$ pwd : shows current directory path. It is short for Print Working Directory.

$ clear : It clears screen.

$ cd : Changes directory. Just cd command changes directory to my home directory.
And cd / changes directory to root directory.
cd .. changes directory to parent directory of current directory.

$ man ls : Print manual about ls command.

$ who am i : shows my id, terminal number, login time, and client ip adress.

$ uname -a : shows operating system version and cpu.

$ date : shows current date and time.

$ id : shows my uid(user id) and gid(group id).

$ groups : shows my group.

$ who : shows current logined user by list.
For example, $ who | grep unix300 will show logined unix300 user.
If unix300 is not logined, this command shows nothing.

$ finger : shows current logined user with user name. It is very similar to who command.

$ w : shows current logined user with tasks they doing. It is also similar to who command.

$ exit : logout at unix server.

$ tty : shows current terminal number.

$ stty -a : shows current terminal status and property like key setting.
For example, stty erase [backspace] command changes deleting one character function key to backspace key.

$ echo hello : print hello on terimnal.
